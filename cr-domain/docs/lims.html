What it validates

ruleapplies toseverityrequires
cr:SpecimenOriginShapehcls:SpecimenViolation
  • Specimen must have an origin: a collection event (cr:collectedAt) or a parent (prov:wasDerivedFrom).
cr:CustodyEventShapecr:CustodyEventViolation
  • Custody event must reference its specimen.
  • Custody event toState must be a valid lifecycle state.
cr:AssayResultShapecr:AssayResultViolation
  • Assay result must join to the (pseudonymous) StudySubject.
  • Assay result must reference the analysis service that produced it.
  • Assay result must reference the sample analyzed.
