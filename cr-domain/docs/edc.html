What it validates

ruleapplies toseverityrequires
cr:ClinicalObservationShapecr:ClinicalObservationViolation
  • Observation must reference its variable definition (DataElementDef).
  • Observation must reference the (pseudonymous) StudySubject.
  • Observation must reference the StudyEvent it was captured at.
cr:DataElementPHIShapecr:DataElementDefViolation
  • DataElementDef should carry a PHI classification (NotPHI/LimitedPHI/FullPHI/Restricted).
cr:QueryShapecr:QueryViolation
  • Query must be anchored to a target entity (cr:queryOn).
  • Query must carry a resolution status (open/answered/closed).
