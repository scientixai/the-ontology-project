ruleapplies toseverityrequires
cr:EnrollmentShapecr:EnrollmentViolation

  • Enrollment must reference the pseudonymous StudySubject.
  • Enrollment must reference the real Person (PII bridge).
cr:InformedConsentShapecr:InformedConsentViolation
  • Consent must record who performed it.
  • Consent must reference the StudySubject.
  • Consent must reference the authorizing Delegation.
